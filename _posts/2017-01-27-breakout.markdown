<h1>What is this?</h1>
A Breakout game made using Craftyjs and Javascript. Objective is to blow up all the bricks. 

When you blow up a brick there is a chance it will drop a PowerUp. If you're able to pick up the PowerUp it will be used instantly and show up on top left.
